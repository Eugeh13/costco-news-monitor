# Screenshots para Claude

Esta carpeta existe para compartir imágenes con Claude en chats de claude.ai
cuando se llega al límite de uploads directos.

## Cómo funciona

1. Sube imagen a esta carpeta (manualmente o con el alias `sshot`)
2. Commitea y pushea
3. Pega la URL raw en el chat

## Formato de URL

Las URLs raw son accesibles públicamente. Formato:

```
https://raw.githubusercontent.com/Eugeh13/costco-news-monitor/main/docs/screenshots/NOMBRE_IMAGEN.png
```

Ejemplo:
```
https://raw.githubusercontent.com/Eugeh13/costco-news-monitor/main/docs/screenshots/dashboard_2026-04-20.png
```

## Convención de nombres

```
DESCRIPCION_YYYY-MM-DD.png
```

Ejemplos:
- `dashboard_2026-04-20.png`
- `pipeline_error_2026-04-21.png`
- `telegram_alert_2026-04-22.png`
